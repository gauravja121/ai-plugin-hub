Authentication

  -H "Content-Type: application/json"
  -d '{"api_key": "your_api_key"}'

Generate Plugin

curl -X POST https://api.pluginforge.ai/generate
  -H "Content-Type: application/json"
  -H "Authorization: Bearer YOUR_TOKEN"
  -d '{
    "description": "A contact form with validation",
    "platform": "wordpress",
    "options": {
      "include_documentation": true
    }
  }'

Frequently Asked Questions

Do I need coding knowledge to use PluginForge AI?

No, our platform is designed for users with no coding experience. Our AI handles the technical aspects, allowing you to focus on what you want your plugin to do.

What platforms are supported?

We currently support WordPress, Shopify, Wix, and custom HTML/JS/CSS for any website. We're constantly adding more platforms based on user feedback.

How secure are the generated plugins?

All plugins are generated with security best practices in mind. We regularly update our systems to ensure the latest security standards are followed. However, we recommend testing plugins thoroughly before deploying to production.

Can I modify the generated code?

Yes, you have full access to the generated code and can modify it using our built-in code editor or after exporting it.

Do I own the plugins I create?

Yes, you retain full ownership of any plugins created using our platform. You can use them for personal or commercial purposes.
